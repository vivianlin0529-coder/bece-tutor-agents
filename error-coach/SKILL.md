# Error Analysis Skill

## Overview
The Error Analysis skill supports students in diagnosing and remedying learning mistakes.

## Instructions
1. Collect data on common errors made by students.
2. Analyze patterns to identify root causes.
3. Provide tailored strategies for overcoming these errors.

## Teaching Formats
- Error Review Sessions
- Individual Tutoring

## Routing Logic
- Direct students with frequent errors to tailored remedial sessions.
- Use group analysis for broader trends among multiple students.

## Examples
- Reviewing previous test results to identify common mistakes.

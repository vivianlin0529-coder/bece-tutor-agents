# Concept Explanation Skill

## Overview
This skill uses concept maps and analogies to enhance understanding and retention of information.

## Instructions
1. Identify difficult concepts for the student.
2. Create visual concept maps to explain relationships.
3. Use analogies to relate concepts to familiar ideas.

## Teaching Formats
- Interactive Lectures
- Workshop Sessions

## Routing Logic
- For complex concepts, prioritize using detailed concept maps.
- Use analogies for students expressing confusion about specific terminology.

## Examples
- Explaining electric circuits using a water flow analogy.

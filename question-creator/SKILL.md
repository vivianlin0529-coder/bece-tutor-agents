# Assessment Design Skill

## Overview
The Assessment Design skill focuses on creating engaging and effective Anki cards for knowledge retention.

## Instructions
1. Determine key learning objectives based on curriculum.
2. Create questions that correspond to those objectives.
3. Ensure cards include spaced repetition factors on review.

## Teaching Formats
- Self-Directed Learning
- Collaborative Study Groups

## Routing Logic
- Target students preparing for exams for targeted card creation.
- Use more general questions for broader review sessions.

## Examples
- Creating a card that asks about the water cycle with audio/visual aid.
